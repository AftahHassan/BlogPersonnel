# 📝 BlogPersonnel — Application de Blog avec Laravel

## 📌 Description du projet

**BlogPersonnel** est une application web développée avec **Laravel** permettant à un blogueur de publier et gérer ses articles en ligne.

Le système distingue deux types d’utilisateurs :
- 👤 **Visiteur** : peut consulter les articles publiés
- ✍️ **Blogueur (authentifié)** : peut gérer ses articles (CRUD)

L'objectif est de respecter les bonnes pratiques Laravel :
- Architecture MVC
- Eloquent ORM
- Blade Templates
- Sécurité et validation

---

## 🚀 Fonctionnalités

### 🌍 Partie publique
- 📄 Liste des articles publiés
- 🔍 Filtrer les articles par catégorie
- 📖 Voir le détail d’un article

### 🔐 Authentification
- Connexion / Déconnexion
- Compte créé via Seeder (pas d'inscription)

### 🧑‍💻 Espace blogueur
- 📊 Tableau de bord (articles brouillons + publiés)
- ➕ Créer un article
- ✏️ Modifier un article
- ❌ Supprimer un article
- 🟡 Gérer le statut (Brouillon / Publié)

---

## 🛠️ Technologies utilisées

- ⚙️ Laravel
